# Product Image Library

Product image assets are shared by the English main site and the `/jp` Japanese subsite.

Each product uses the same structure:

```text
assets/products/<product-slug>/
  cover/      Primary product cover image
  details/    Product detail and close-up images
```

Current product folders:

- `linkcore-01` - HDMI to USB-C Adapter
- `fiberline-02` - USB-C to USB-C Cable
- `netlink-03` - USB-C Ethernet Adapter
- `bridge-04` - USB-A to USB-C Adapter

Recommended filenames are `cover.webp` for the main image and `detail-01.webp`, `detail-02.webp`, and so on for close-up images.
